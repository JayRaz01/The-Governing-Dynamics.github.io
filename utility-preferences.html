Utility and Preferences

Utility is an outcome of actions (our decisions). We can think of the total set of actions \(A\) as containing all possible actions we could take, represented by \( a_i, i = 1,2,3, \dots \).

When we take these actions, we obtain an outcome known as the utility of those actions.

To quantify these outcomes mathematically, we use numbers (e.g., \( 1,2,3,4,\dots \)), possibly including negative numbers to represent disutility.

Since utility is measured in regular numbers, we define this set mathematically as the set of real numbers \( \mathbb{R} \).

The Utility Function

Actions from \(A\) produce outcomes in \( \mathbb{R} \). The function determining which actions generate which utilities is known as the utility function.

Mathematically, we define the utility function as:

\[ U: A \to \mathbb{R} \]

Example: Bob Eating Chocolate

Bob enjoys chocolate and decides how many bites to eat. His action set is:

\[ A = \{ a_0, a_1, a_2, \dots, a_8 \} = \{ 0,1,2,\dots,8 \} \]
